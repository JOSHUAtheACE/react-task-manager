A simple and responsive Task Manager application built using React. It helps users manage daily tasks by allowing them to add, edit, delete, and organize tasks with different priorities and statuses.


Concepts involved:

1. Components
2. UseState and UseEffect
3. Conditional Rendering
4. Rendering Lists using .map
5. Js basic concepts
6. Event Handling
7. State Management
8. props
